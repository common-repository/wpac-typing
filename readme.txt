Now you can write Urdu Unicode in your WordPress Posts or Pages.

== Description ==

This Urdu Typing plugin will let you write Urdu in WordPress. You can write Urdu for Post Title, Post Content, Category Names & Tags. And thats all with a great Urdu font Nafees Nastaleeq. You don't have to install any font or keyboard in your system.

This plugin is compatible with Cklassic Editor only at the moment, so make sure you have classic editor plugin installed and activated.

= Credits =
This plugin is based on [Urdu Keyboard]( https://wordpress.org/plugins/urdu-keyboard/ "Urdu Keyboard") orginally developed by @zayedbaloch.
Urdu Keyboard JS Library by [UrduWeb](https://github.com/urduweb/urdueditor/tree/master/jquery)


== Frequently Asked Questions ==

= How to Install the Plugin =

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WPAC Settings screen to configure the plugin

= Compatible Browsers =
Chrome, Firefox, IE10, IE11, Safari and Opera.

= How to get help =

You can post into our [Forums]( https://wpacademy.pk/forum "WPAC Plugin Help Forum")

= How to change keyboard language? =

Press `Ctrl + Space` to change keyboard writing from `English to Urdu` or `Urdu to English`.
